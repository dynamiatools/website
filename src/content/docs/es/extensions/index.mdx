- proyectos backend (Maven/Gradle),
- aplicaciones frontend (paquetes Node), cuando aplique.

## Nota de version

Los ejemplos usan `{dynamiaToolsVersion}` porque es la version actualmente referenciada en este monorepo.

Si tu proyecto centraliza la version de Dynamia, reemplaza valores hardcodeados por tu propiedad de version.
