This section documents the official Dynamia Platform extensions and how to integrate them in:

- backend projects (Maven/Gradle),
- frontend apps (Node packages), when available.

## Available extensions

- [Dashboard](./dashboard/)
- [Email & SMS](./email-sms/)
- [Entity Files](./entity-files/)
- [File Importer](./file-importer/)
- [Finances](./finances/)
- [HTTP Functions](./http-functions/)
- [Reports](./reports/)
- [SaaS](./saas/)
- [Security](./security/)

## Not listed here

MicroFrontend integration ships as a core framework capability (`tools.dynamia.zk`), not a Maven module under `extensions/`, so it's documented under Guides instead: see [MicroFrontends](/guides/microfrontends).

## Versioning note

Examples use `{dynamiaToolsVersion}` because it is the version currently referenced in this monorepo.

If your project centralizes the Dynamia version, replace hardcoded values with your own version property.
